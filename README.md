# LinuxHomeLab

## OVERVIEW
Linux Server Home Lab is a project to simulate the environment of a linux server by using Ubuntu distro. 
This project is made to be a personal practice project to learn and understand more about being a system administrator.
This project contains a few things such as : 
- Installation of Ubuntu
- User & Group management on linux
- SSH Configuration
- Nginx Web Server

## Objectives
This Home Lab has a few key objective such as : 
- Install a secure and usable Ubuntu as a server.
- Use a virtual machine to install the linux.
- Practice Linux command and Linux administration.
- Learn how to do a SSH into Linux server.
- Learn to deploy a web server.
- Document every steps and configuration.

## Environment 
- OS : Ubuntu Server 26.04 LTS
- Virtual Machine : VMWare
- RAM : 4 GB
- CPU : 2 Core
- Storage : 30 GB
